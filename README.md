# Библиотека для комплексных чисел

Самописная библиотека для комплексных чисел. Проект где я вникаю в CMake и в gtest.
Решил просто разобраться в том как люди пишут код (много кода) на c++, делают код расширяемым и поддерживаемым.

Так же доступна <a href="https://swinger365.github.io/ComplexNumbers/">документация</a> (тоже пробую в первый раз). Проект направлен не столько на код, сколько на понимание системы сборки.

Возможно буду использовать эту библиотеку в дальнейшем в своих проектах (планирую рисовать множество Мандельброта).
